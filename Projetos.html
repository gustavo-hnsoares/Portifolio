appLivraria

Aplicativo de Gestão de Livraria para Desktop

Este projeto envolve o desenvolvimento de um aplicativo de gestão de livraria para desktop, atualmente em fase de desenvolvimento. O design das telas foi criado utilizando o Adobe XD, proporcionando uma interface visualmente atraente e fácil de usar. O desenvolvimento do aplicativo está sendo realizado em Java, com o uso do JavaFX para a criação das telas interativas.

Para a comunicação com o banco de dados, estão sendo utilizados JDBC e JTDS, garantindo a eficiência nas requisições e manipulação de dados. O projeto segue o padrão de arquitetura MVC (Model-View-Controller), promovendo uma separação clara entre a lógica de negócios, a interface do usuário e o controle de fluxo.

O aplicativo possui sete telas principais:

  • Cliente: Gerenciamento de informações dos clientes.
  • Editora: Controle de dados das editoras parceiras.
  • Funcionário: Administração dos dados dos funcionários.
  • Home: Tela inicial com visão geral do sistema.
  • Login: Autenticação de usuários.
  • Livro: Gestão do estoque de livros.
  • Pedido: Controle de pedidos

O aplicativo é desenvolvido em Java, utilizando o JavaFX para a interface gráfica. O banco de dados utilizado é o SQL Server, com acesso via JDBC e JTDS. O projeto segue o padrão MVC (Model-View- Controller), garantindo uma estrutura organizada e de fácil manutenção.

O projeto está em andamento e, assim que concluído, será disponibilizado no GitHub.

appLista_Tarefas

Aplicação de Gestão de Tarefas

Este projeto consiste no desenvolvimento de uma aplicação para gestão de tarefas, utilizando tecnologias frontend e backend. A aplicação é baseada em uma playlist do canal João Ribeiro, com o objetivo de aprender e aplicar conceitos de desenvolvimento web.

Tecnologias Utilizadas: HTML, CSS, JavaScript puro, Bootstrap no frontend; Node.js, Express, MySQL no backend.

Estrutura do Banco de Dados:

  • Tabela: users
    • id
    • username
    • password
    • created_at
    • updated_at
  • Tabela: tasks
    • id
    • id_user
    • task_text
    • task_status (new | in progress | canceled)
    • created_at
    • updated_at

Tarefas que foram desenvolvidas no Projeto:

  • Criar a Estrutura Inicial:
    • Configurar a base do frontend (HTML, CSS, JS, Bootstrap)
    • Configurar a base do backend (Node.js, Express, MySQL) com uma resposta padrão
  • Frontend:
    • Desenvolver as páginas necessárias para a navegação no aplicativo
    • Realizar pequenos testes de comunicação entre frontend e backend utilizando Ajax (XMLHttpRequest | Fetch API)
  • Estrutura Base de Cada Página:
    • Utilizar Bootstrap (tema Slate do Bootswatch)
    • Integrar FontAwesome para ícones
  • Ver Tarefas:
    • Exibir título
    • Implementar filtro para selecionar as tarefas a serem visualizadas (select)
    • Adicionar botão para criar novas tarefas
    • Exibir mensagem caso não existam tarefas
    • Criar caixa para exibição das tarefas, com opções para alterar status, editar e excluir tarefas
    • Exibir parágrafo com o total de tarefas disponíveis (de acordo com o filtro)
  • Adicionar Tarefa:
    • Campo de texto para inserir a descrição da tarefa
    • Botão para cancelar
    • Botão para submeter a nova tarefa
  • Editar Tarefa:
    • Campo de texto para editar a descrição da tarefa
    • Botão para cancelar
    • Botão para submeter a alteração
  • Excluir Tarefa:
    • Implementar exclusão de tarefas utilizando uma modal

Este projeto visa proporcionar uma experiência prática no desenvolvimento de uma aplicação completa, desde a criação da interface de usuário até a implementação do backend e integração com o banco de dados.

O projeto está em andamento e, assim que concluído, será disponibilizado no GitHub.
